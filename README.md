# Seats — Classroom Seat Assignment Manager

> A modern, privacy-first classroom seat assignment tool. Assign students to seats, manage aisles, handle bulk operations, and export to Excel — all running entirely in your browser with zero third-party tracking. Now featuring a **smart seating engine** that auto-generates seat lists from exam scores with multi-tier privilege rules.

## Features

### Seat Arranger (`/`)

| Feature | Description |
| ------- | ----------- |
| **Click-to-assign** | Click a name, then click a cell — no drag needed. Or use random fill for instant distribution |
| **Aisle management** | Mark columns as aisles; displaced students are automatically returned to the list |
| **Swap & remove modes** | Two dedicated modes with clear visual feedback for safe editing |
| **Bulk operations** | Drag-select or click twice to batch-toggle seat availability |
| **Excel export** | Download your seat map as a `.xlsx` file ready for printing |
| **Import from Engine** | Seamlessly import seating results generated by the Smart Seating Engine |
| **Multi-language** | Switch between English and Chinese (简体中文) with one click; persists in `localStorage` |
| **Offline capable** | Pure client-side SolidJS app; works offline after first load |
| **Privacy-first** | Zero third-party tracking, no server required |

### Smart Seating Engine (`/engine`)

| Feature | Description |
| ------- | ----------- |
| **XLSX Score Import** | Upload a multi-sheet workbook: one sheet for the current exam, one for the previous exam. Automatically parses rankings and scores |
| **Group / Guild System** | Students can be grouped into teams/guilds. Upload a custom `.json` group registry or use the default hash-based grouping |
| **Tiered Privilege Rules** | Auto-selects students based on configurable rules: class top N, math school top N, subject-specific quotas, and math score thresholds |
| **Progress / Comeback Mode** | Calculates weighted progress scores from rank improvements across exams. Supports "all students" or "top N comebacks" |
| **Best Group Bonus** | Automatically grants seats to all members of the highest-average-score group |
| **Lottery Fallback** | Any remaining students are placed into a fair lottery batch |
| **Export & Import** | Export results as `.json` or import them directly back to the Seat Arranger page |

## Usage

### Seat Arranger (`/`)

The main grid displays a **10×10** table of seats. Each cell can be in one of these states:

| State | Visual | Description |
| ----- | ------ | ----------- |
| **Empty** | White/light with coordinate label | Available for assignment |
| **Filled** | Blue tinted | Contains a student name |
| **Disabled** | Grayed out | Permanently unavailable |
| **Aisle** | Dark column | Represents a walkway, no seats |

#### Modes

Use the **mode toggle buttons** at the top of the sidebar to switch between:

- **Fill (Sit Down)**: Assigning and swapping students
- **Remove (Yeet)**: Removing students from seats

#### Assigning Students

1. Click a name in the **right sidebar list** to select it
2. Click any **empty cell** in the grid to assign that student
3. The name disappears from the list after assignment

#### Swapping Two Students

1. In **Fill** mode, click a **filled cell** — it highlights as "swap selected"
2. Click another **filled cell** to swap the two students
3. A brief flash animation confirms the swap

#### Removing a Student

1. Switch to **Remove** mode (the button turns red)
2. Click a **filled cell** — it turns red to confirm
3. Click again to confirm removal; the name returns to the list

#### Managing Aisles

Click the column buttons above the grid to toggle a column as an **aisle**:

- Aisles cannot hold students
- Any student in that column is displaced back to the list
- The column button turns dark to indicate aisle status

#### Bulk Enable/Disable Seats

In **Fill** mode with no name selected:

1. Click an **empty cell** to start a batch selection
2. Click another cell to define a rectangle — all empty cells in that area highlight
3. Click once more to **confirm and disable** all highlighted cells

To re-enable disabled seats:

1. Click a **disabled cell** — it highlights in amber
2. Click another cell to define the batch area
3. Click again to **confirm and enable** all cells

#### Random Fill

Click **Random Fill** in the sidebar to automatically distribute all names in the list to random available seats.

#### Import from Engine

After generating a result in the **Smart Seating Engine** (`/engine`), click **Import result to Seat Arranger** to transfer the generated name list (with category dividers) directly to the sidebar.

You can also upload a `.json` engine result file via the file input on the home page.

#### Export to Excel

Click **Export XLSX** to download the current seat map as a `.xlsx` file.

#### Language Switch

Click the **language button** (EN / 中文) in the sidebar header to toggle between English and Chinese. Your preference is saved to `localStorage`.

### Smart Seating Engine (`/engine`)

The engine generates a prioritized student list from exam score data using a multi-tier rule system.

#### Data Input

1. Upload an `.xlsx` workbook with at least two sheets:
   - **Current Exam Sheet**: columns = `Name | Total Score | School Rank | (Class Rank) | Chinese/Rank | Math/Rank | English/Rank | Physics/Rank | Chemistry/Rank | Biology/Rank` *(Class Rank is optional — the engine will auto-calculate it from School Rank if omitted)*
   - **Previous Exam Sheet**: columns = `Name | Total Score | School Rank | Chinese/Rank | Math/Rank | ...`
2. Select which sheet represents the current exam and which represents the previous exam
3. (Optional) Upload a `.json` group registry to override default group assignments

#### Rule Configuration

| Setting | Description |
| ------- | ----------- |
| **Class Top N** | Number of top class-ranked students granted automatic privilege seats |
| **Math School Top N** | Number of top math school-ranked students granted automatic privilege seats |
| **Math Score Threshold** | Minimum math score for an additional "exceptional math" privilege seat |
| **Subject Quota** | Per-subject privilege quota (for Chinese, English, Physics, Chemistry, Biology) |
| **Progress Mode** | `All` — all students with rank improvement are eligible; `Top N` — only top N by weighted progress score |
| **Progress Weights** | Configurable point values per rank interval (e.g., 401+, 201–400, 101–200, 11–100, 6–10, 1–5) |

#### Result Batches

The engine outputs students in four prioritized batches:

1. **Privilege (The Chosen)** — students selected by class rank, math rank, subject quotas, or math score threshold
2. **Progress (Comeback Heroes)** — students with the highest weighted progress scores from rank improvement
3. **Group (Strongest Guild)** — all unselected members of the highest-average-score group
4. **Lottery (Wheel of Fate)** — all remaining students

#### Actions

- **Import to Seat Arranger**: Transfers the result list (with batch dividers) to the home page
- **Download JSON**: Exports the full engine result as a timestamped `.json` file
