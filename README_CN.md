# Seats — 教室座位分配管理工具

> 一个现代化的、注重隐私的教室座位分配工具。在浏览器中完成学生座位分配、过道管理、批量操作和 Excel 导出，无任何第三方追踪。现已搭载**智能排座引擎**，可根据考试成绩数据与多层特权规则自动生成座位名单。

## 功能特点

### 座位编排器（`/`）

| 功能 | 说明 |
| ---- | ---- |
| **点击式分配** | 点击姓名，再点击空格，即可填入。也可一键随机分配 |
| **过道管理** | 将整列标记为过道，被清退的学生自动归还名单 |
| **交换/移除模式** | 两种独立模式，视觉反馈清晰，操作安全 |
| **批量管理** | 框选或双击即可批量启用/禁用座位 |
| **Excel 导出** | 将座位表导出为 `.xlsx` 文件，直接打印使用 |
| **从引擎导入** | 一键将智能排座引擎生成的结果导入到座位编排器 |
| **多语言支持** | 一键切换中英文；语言偏好自动保存到 `localStorage` |
| **离线可用** | 纯前端 SolidJS 应用，首次加载后可离线使用 |
| **隐私优先** | 零第三方追踪，无需服务器 |

### 智能排座引擎（`/engine`）

| 功能 | 说明 |
| ---- | ---- |
| **XLSX 成绩导入** | 上传包含多个 Sheet 的工作簿：一个用于本次考试，一个用于上次考试。自动解析排名与分数 |
| **分组/公会系统** | 学生可分组为战队/公会。支持上传自定义 `.json` 公会名册，或使用默认哈希分组 |
| **分层特权规则** | 根据可配置规则自动选座：班级 Top N、数学年级 Top N、各学派特权名额、数学分数线 |
| **逆袭/进步模式** | 根据两次考试的年级排名变化计算加权逆袭积分。支持「全员逆袭」或「Top N 逆袭者」 |
| **最强公会福利** | 自动为平均分最高的公会的所有未选座成员授予座位 |
| **命运兜底抽签** | 其余未触发任何规则的学生进入公平的命运轮盘抽签 |
| **导出与导入** | 可将结果导出为 `.json` 文件，或直接导入回座位编排器页面 |

## 使用说明

### 座位编排器（`/`）

主网格显示 **10×10** 的座位表格，每个单元格可能处于以下状态：

| 状态 | 视觉效果 | 说明 |
| ---- | -------- | ---- |
| **空座位** | 白色/浅色，显示坐标标签（如 A1） | 可分配 |
| **已填充** | 蓝色调 | 已有学生姓名 |
| **已禁用** | 灰色 | 永久不可用 |
| **过道** | 深色整列 | 表示过道，无座位 |

#### 模式切换

使用侧边栏顶部的**模式切换按钮**在两种模式间切换：

- **入座**：用于分配和交换学生
- **驱逐**：用于从座位移除学生

#### 分配学生

1. 点击**右侧名单**中的学生姓名，将其选中
2. 点击网格中任意**空单元格**，该学生即被分配到该座位
3. 分配成功后，姓名从名单中消失

#### 交换两个学生

1. 在**入座**模式下，点击一个**已填充单元格** — 该座位高亮显示「交换选中」状态
2. 点击另一个**已填充单元格**，两位学生互换位置
3. 短暂的闪烁动画确认交换完成

#### 移除学生

1. 切换到**驱逐**模式（按钮变为红色）
2. 点击一个**已填充单元格** — 该座位变红提示确认
3. 再次点击以确认移除，学生姓名回到名单

#### 管理过道

1. 点击网格上方的列按钮，可将该列切换为**过道**
2. 过道内如有学生，会被自动清退并归还到名单
3. 列按钮变深色表示该列已设为过道

#### 批量启用/禁用座位

**禁用座位**（在入座模式下且未选中姓名时）：

1. 点击一个**空单元格**开始框选
2. 点击另一个单元格确定矩形范围，所有空单元格高亮
3. 再次点击确认，将所有高亮单元格**设为禁用**

**启用座位**：

1. 点击一个**已禁用单元格**，高亮显示
2. 点击另一单元格确定范围
3. 再次点击确认，将所有高亮单元格**设为启用**

#### 随机填入

点击侧边栏中的**随机召唤**，可将名单中的所有学生一键随机分配到空白座位。

#### 从引擎导入

在**智能排座引擎**（`/engine`）生成结果后，点击**将炼成结果导入座位编排**即可将生成的名单（含分类分隔符）直接传送到侧边栏。

你也可以在首页通过文件上传框直接上传引擎生成的 `.json` 结果文件。

#### 导出 Excel

点击**导出 XLSX**，将当前座位表下载为 `.xlsx` 文件。

#### 语言切换

点击侧边栏标题旁的**语言按钮**（EN / 中文）切换中英文界面。语言偏好会自动保存到 `localStorage`。

### 智能排座引擎（`/engine`）

引擎根据考试成绩数据，通过多层规则系统生成一份带有优先级的学生座位名单。

#### 数据录入

1. 上传一个至少包含两个 Sheet 的 `.xlsx` 工作簿：
   - **本次考试 Sheet**：列格式为 `姓名 | 总分 | 校次 | (班次) | 语文/校次 | 数学/校次 | 英语/校次 | 物理/校次 | 化学/校次 | 生物/校次`（班次可选，省略时会根据校次自动计算）
   - **上次考试 Sheet**：列格式为 `姓名 | 总分 | 校次 | 语文/校次 | 数学/校次 | ...`
2. 选择哪个 Sheet 代表本次考试，哪个代表上次考试
3.（可选）上传 `.json` 公会名册以覆盖默认分组

#### 规则配置

| 配置项 | 说明 |
| ------ | ---- |
| **班级精英 Top N** | 班级排名前 N 的学生自动获得天选之人的特权座位 |
| **炼金·数学全服 Top N** | 数学年级排名前 N 的学生自动获得特权座位 |
| **炼金·数学分数线** | 数学分数达到该分数线以上的学生获得额外「极优」特权座位 |
| **各学派特权名额** | 每门学科（语文、英语、物理、化学、生物）的特权名额 |
| **逆袭模式** | `全员逆袭` — 所有有进步的学生均可参与；`Top N 逆袭者` — 仅积分最高的前 N 名 |
| **积分权重配方** | 可配置的各排名区间逆袭积分（如 401+、201–400、101–200、11–100、6–10、1–5） |

#### 结果批次

引擎按以下四个优先级批次输出学生：

1. **天选之人** — 通过班级排名、数学排名、各学派特权名额或数学分数线选中的学生
2. **逆袭勇者** — 逆袭积分最高的学生（根据两次考试的年级排名变化计算）
3. **最强公会** — 平均分最高公会中尚未选座的所有成员
4. **命运轮盘** — 其余所有未触发任何规则的学生

#### 操作

- **将炼成结果导入座位编排**：将结果名单（含批次分隔符）传送到首页
- **下载炼金配方 (.json)**：将完整引擎结果导出为带时间戳的 `.json` 文件

- **框架**: SolidJS + `@solidjs/router`

- **Excel 读写**: ExcelJS（导入与导出）
